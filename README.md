# algostudy

## clone 받기 전 간단한 세팅

커맨드 라인에 입력하여 디폴트 세팅해주세요

```
    git config --global core.editor vim
    git config --global user.name "자기 이름"
    git config --global user.email "자기 깃허브 이메일주소"
```

레포지토리 다운로드

```
 git clone https://github.com/dsahn/algostudy.git
```
